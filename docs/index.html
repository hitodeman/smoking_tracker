あなたの禁煙をサポートするアプリ

📱 アプリについて

「ゼロモク」は、喫煙習慣を記録し、禁煙目標の達成をサポートするためのシンプルなアプリです。 毎日の喫煙本数を記録し、視覚的なグラフで進捗を確認できます。

✨ 主な機能

  • 日々の喫煙本数を簡単に記録
  • 美しいグラフで喫煙データを可視化
  • 月間統計で進捗を確認
  • 喫煙にかかる費用を自動計算
  • 目標設定で禁煙をサポート
  • 完全オフライン動作 - データは全てデバイス内に保存

🔒 プライバシー

「ゼロモク」は、あなたのプライバシーを最優先に考えています。 全てのデータはあなたのデバイス内にのみ保存され、外部サーバーへの送信は一切行いません。
